# Generated Samples

This folder contains sample AI-generated product photography outputs produced by TaskHub.

## Expected Output Structure (per product)

```
generated_samples/
├── sample_ring/
│   ├── 01_white_background.jpg     — Pure white studio background
│   ├── 02_theme_background_a.jpg   — Lifestyle/contextual scene
│   ├── 03_theme_background_b.jpg   — Alternative theme setting
│   ├── 04_creative_a.jpg           — Artistic editorial treatment
│   ├── 05_creative_b.jpg           — Dramatic composition
│   ├── 06_model_front.jpg          — Model wearing, front view
│   ├── 07_model_side.jpg           — Model wearing, 45° angle
│   └── 08_model_closeup.jpg        — Detail close-up on model
└── README.md
```

## Product Consistency Requirements

For all 8 outputs, the following MUST remain identical:
- Product shape and silhouette
- Color and metal finish
- Stone/gem characteristics
- Design details and engravings
- Proportions and scale

Only these elements change between outputs:
- Background / scene / environment
- Lighting mood and direction
- Camera angle (for model shots)
- Human model (for wearing shots)

## Generation Quality Targets

- Resolution: 1024×1024 minimum
- Quality: DSLR / professional e-commerce level
- No AI artifacts, distortions, or blurriness
- Photorealistic, not cartoon or illustrated
- Suitable for direct use in product listings
